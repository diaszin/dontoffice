# Requisitos

## Requisitos Funcionais
- O usuário deve criar uma rota com qualquer nome. Ex: dontoffice.com/miscamusca
- O usuário poderá importar um arquivo docx ou md.  
- O usuário poderá realizar o download do arquivo importado na url
- O usuário deve visualizar o arquivo importado(Preview), se não houver nada, deve ficar em branco
- O usuário poderá compartilhar a rota criada

## Requisitos Não Funcionais
- A aplicação deve ser responsiva
- A aplicação deve ser rápida
- A aplicação deve ser segura
- A aplicação deve ser acessível
- A aplicação deve ser visualmente parecida com o [Dontpad](https://dontpad.com/)
