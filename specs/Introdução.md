O projeto dontoffice é um hub de ferramentas de escritório que tem como objetivo facilitar o compartilhamento de informações, não sendo exclusivamente necessário uso de login ou cadastro para usar as ferramentas disponíveis.

Os principais módulos/funcionalidades devem espelhar as ferramentas de escritórios já inseridas no mercado, como o Microsoft Office, Google Docs ou LibreOffice.
