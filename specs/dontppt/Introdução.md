O dontppt é a ferramenta de apresentação de slide. O objetivo principal não é a edição de slides e sim a apresentação dele.

## Requisitos não funcionais
1. O sistema deverá rodar nos principais navegadores
2. O sistema deverá permitir o compartilhamento de link

## Requisitos funcionais
1. O usuário deverá definir uma rota para o seu slide
2. A rota deverá seguir o padrão das URLs modernas. Sendo elas: letras minúsculas, substituição de hífen por espaço e não deve conter caracteres especiais
3. o sistema deverá permitir somente a importação de slides que contenha as seguintes extensões: .ppt, .pps, .pptx, .ppsx e .odt
4. O usuário poderá visualizar o slide importado
5. O usuário poderá alterar o slide importado
6. O usuário poderá avançar os slides de um 1 a 1
7. O usuário poderá retroceder os slides de um 1 a 1
8. O usuário poderá permitir a visualização automática dos slides
9. O usuário deverá inserir o tempo de avanço dos slides para a visualização automática dos slides
10. o sistema deve permitir a importação do slide através da técnica drag-and-drop
11. O usuário poderá pesquisar rotas criadas no sistema
