# Don't office

Projeto inspirado no [Dontpad](https://dontpad.com/)

## Descrição

O Don't office é um armazenamento de arquivo docx e md open url, onde você pode criar um link e compartilhar com quem quiser. Sem a necessidade de colocar em ambientes de nuvem como Google Drive ou OneDrive para compartilhar arquivos. O arquivo é salvo em um banco de dados local e pode ser acessado por qualquer pessoa com o link.
