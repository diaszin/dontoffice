# Dontoffice

---

O Dontoffice é um projeto que tem as ferramentas de escritórios como Microsoft Office ou Google Suits, porém open source e online.



## Design

O design da aplicação e outros fluxogramas necessários será feito no [Penpot ](https://help.penpot.app)(localhost).  A ideia é facilitar a criação de designs, mantendo os criativos e fluxogramas em um único local
